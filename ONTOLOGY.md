# Ontology

Shared language reduces drift.

## Core Terms

### Akasha
The ecosystem / constellation as a whole.

### Axioms
The highest-level invariants of the ecosystem.

### World
The constitutional law layer that defines system structure.

### Table
The classification layer that defines types, roles, or structural positions.

### Constellation
The living registry and relationship map of all systems.

### System
Any declared engine, forge, node, device, academy, runtime, adapter, graph, canon, or lab.

### Canonical
Aligned, declared, placed, and admitted into the ecosystem.

### Lab
An experimental system not yet admitted as canonical.

### Stewardship
Human accountability for governance, review, and maintenance.

### Discovery
Structured inference of missing knowledge or missing system components.

### Tool Creation
The generation of new instruments when discovery requires them.

### Runtime
The layer that governs executable behavior, invariants, and failure containment.
