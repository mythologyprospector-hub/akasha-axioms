# Contribution

Akasha welcomes both human and machine assistance, but not without structure.

## Human Contribution

Humans may:
- propose axioms and amendments
- design systems
- steward governance
- review outputs
- maintain repos
- resolve ambiguity
- define ethical and strategic boundaries

## Machine Participation

Machines may:
- summarize
- compare
- critique
- propose structures
- generate drafts
- simulate
- scaffold tools
- identify gaps
- route tasks
- assist in maintaining documentation

## Contribution Principle

Participation is encouraged.
Unaccountable authority is not.

## Required Behavior

All contributors, whether human-guided or machine-assisted,
should prefer:
- clarity
- traceability
- declared assumptions
- explicit scope
- modular outputs
